    XVerbal announcements
Text-to-speech using the browser's built-in speech synthesis — "Round starting in 3… 2… 1…", "Rest", "Last round" — so instructors don't have to watch the screen.

Display themes
Let instructors pick a color scheme or background image for the display. Some gyms want their brand colors, dark vs light, etc.

Scheduled class mode
Pre-configure a full class: 10-minute warmup → 5 rounds of 5 minutes → cooldown. Hit one button and it runs the whole sequence automatically.

    XRemote display auto-reconnect
If the server restarts, displays currently just sit disconnected. Auto-reconnect with exponential backoff would make TVs self-healing.

    XPIN-per-controller instead of one shared password
Right now all 4 controllers share one password. Individual PINs per instructor slot would be more secure and let you manage who can access what.

XTablet landscape lock
The controller UI is portrait-optimized for phones but tablets held landscape would benefit from a two-column layout — timer big on the left, controls on the right.

    XOffline mode
If the server goes down mid-class, displays currently freeze. A service worker could cache the last known state and show a "server offline" banner rather than a blank screen.
